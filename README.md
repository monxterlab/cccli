# CCCLI - Claude Code 配置管理工具

CCCLI 是一个用于管理 Claude Code 多环境配置的命令行工具。它可以帮助你轻松切换不同的 API 环境（如 Kimi、OpenAI 等），管理认证密钥和代理设置。

## 功能特性

- **多环境配置管理** - 轻松创建和管理多个 Claude Code 配置环境
- **一键切换** - 快速激活指定配置，自动写入 Claude Code 配置文件
- **快速配置** - 使用模板快速创建常用环境配置
- **智能补全** - 支持 Shell 自动补全，减少输入错误
- **代理管理** - 便捷设置系统代理环境变量
- **配置查看** - 随时查看当前配置和环境变量
- **配置删除** - 删除不再需要的配置
- **跨平台支持** - 支持 Windows、macOS 和 Linux

## 安装

### 前置要求

- Node.js >= 16.0.0
- npm 或 yarn

### 全局安装

```bash
# 安装依赖
npm install -g @monxterlab/cccli

```
